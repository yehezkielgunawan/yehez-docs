# Standard Version

A utility for versioning using semver and CHANGELOG generation powered by Conventional Commits.

> âš Please read the full docs [here](https://www.npmjs.com/package/standard-version) before use this.

## Frequent Used Commands

- Default Command

```bash
yarn release
```

- To generate only the CHANGELOG.md

```bash
npx standard-version --skip.bump --skip.commit --skip.tag
```

- To generate the tag versioning manually. You can change the [minor] with your own version like `1.1.0`.

```bash
npm run release -- --release-as [minor]
```
